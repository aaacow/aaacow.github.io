5.可变函数（变量函数）

PHP 中支持可变函数的概念（也叫变量函数），可以这样理解，如果一个变量名后有小括号( )，那么 PHP 就会寻找与变量值同名的函数并执行它。也就是说如果给一个变量赋不同的值，程序就会调用不同的函数。
