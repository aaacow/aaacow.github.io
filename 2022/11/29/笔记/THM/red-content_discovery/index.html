robosts.txt

robots.txt，它告诉搜索引擎它们允许和不允许在其搜索引擎结果中显示哪些页面，或者完全禁止特定搜索引擎抓取该网站。限制某些网站区域可能是常见的做法，因此它们不会显示在搜索引擎结果中。

favicon

favicon 是一个小图标，显示在浏览器的地址栏或选项卡中，用于标记网站。

有时，当框架用于构建网站时，作为安装一部分的网站图标会被遗留下来，如果网站开发人员没有将其替换为自定义图标，这可以让我们了解正在使用的框架。

当网站用框架搭建时，图标作为安装一部分会被遗留下来，如果网站开发者没有将其替换为自定义图标，这可以让我们了解正在使用的框架。

可以使用下面的网站来搜索目标图标 https://wiki.owasp.org/index.php/OWASP_favicon_database。该网站是用图标的md5进行搜索的，所以要先算出图标的md5值。

图标文件我们有时可以在网页源代码里看到：

点开后下载保存，算出md5，再到网站上对比一下即可。

sitemap.xml

与限制搜索引擎爬虫可以查看的内容的 robots.txt 文件不同，sitemap.xml 文件列出了网站所有者希望在搜索引擎上列出的每个文件。这些有时可能包含网站的区域，这些区域更难以导航，甚至列出当前网站不再使用但仍在幕后工作的一些旧网页。

HTTP 标头

curl的使用:https://www.ruanyifeng.com/blog/2019/09/curl-reference.html

当我们向 Web 服务器发出请求时，服务器会返回各种 HTTP 标头。这些标头有时可能包含有用的信息，例如网络服务器软件以及可能使用的编程/脚本语言。
